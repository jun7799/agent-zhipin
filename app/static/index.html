注册 注册用户（按账户限流）

每日可调用：10 次

可用接口：搜索岗位（携带 JWT Token）

额外权益：可保存搜索记录、收藏岗位

第一步：注册

curl -X POST "{{BASE_URL}}/v1/applicant/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "password": "your_password"
  }'

第二步：登录获取 Token

curl -X POST "{{BASE_URL}}/v1/applicant/login" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "password": "your_password"
  }'

// 返回 token，后续请求携带此 token

第三步：携带 Token 查询岗位

curl "{{BASE_URL}}/v1/jobs/search?city=Beijing" \
  -H "Authorization: Bearer <your_jwt_token>"

第一步 企业认证（注册即通过）

审核方式：注册即自动审核通过，立即获得 API Key

返回内容：API Key（ak_开头），后续所有操作都需要此 Key

注册招聘方：

curl -X POST "{{BASE_URL}}/v1/employer/register" \
  -H "Content-Type: application/json" \
  -d '{
    "company_name": "我的公司名称",
    "credit_code": "91110000MA00XXXXXX",
    "email": "hr@company.com",
    "password": "your_password"
  }'

返回：

{
  "success": true,
  "data": {
    "api_key": "ak_xxxxxxxxxxxxxxxx",
    "status": "approved",
    "message": "注册成功，API Key已发放"
  }
}

注意：credit_code 为18位统一社会信用代码，由数字和大写字母组成

请妥善保管 api_key，丢失可通过登录接口重新获取

免费 基础使用（免费6个岗位）

免费额度：6 个岗位

认证方式：Authorization: Bearer <api_key>

可用操作：发布/更新/删除/过期岗位

发布岗位：

curl -X POST "{{BASE_URL}}/v1/employer/jobs" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ak_xxxxxxxxxxxxxxxx" \
  -d '{
    "title": "Python Developer",
    "city": "Shanghai",
    "salary_min": 15000,
    "salary_max": 25000,
    "job_type": "fulltime",
    "description": "岗位职责和要求...",
    "contact_email": "hr@company.com",
    "tags": ["Python", "后端开发"]
  }'

其他操作：

# 查看我的岗位列表
GET  {{BASE_URL}}/v1/employer/jobs
Header: Authorization: Bearer ak_xxx

# 更新岗位
PUT  {{BASE_URL}}/v1/employer/jobs/{job_id}

# 删除岗位（软删除）
DELETE {{BASE_URL}}/v1/employer/jobs/{job_id}

# 设为过期
PATCH {{BASE_URL}}/v1/employer/jobs/{job_id}/expire
充值 充值使用（1元/条）

额外岗位：1 元/条，按需购买

支付方式：支付宝

到账方式：支付成功后自动到账

支付流程：

创建订单 → 获取支付链接 → 支付宝付款 → 额度自动到账

购买岗位额度：

curl -X POST "{{BASE_URL}}/v1/payment/job-slots" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ak_xxxxxxxxxxxxxxxx" \
  -d '{
    "quantity": 10
  }'

// 返回支付链接，金额 = quantity x 1元

查询订单状态：

curl "{{BASE_URL}}/v1/payment/orders/{order_no}" \
  -H "Authorization: Bearer ak_xxxxxxxxxxxxxxxx"
