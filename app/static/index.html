快速开始

1

注册招聘方，获取 API Key

注册成功后自动发放 ak_ 前缀的 API Key，立即可用。

请求：

curl -X POST {{BASE_URL}}/v1/employer/register \
  -H "Content-Type: application/json" \
  -d '{
    "company_name": "我的公司",
    "credit_code": "91110000MA00XXXXXX",
    "email": "hr@example.com",
    "password": "your_password"
  }'

响应：

{
  "success": true,
  "data": {
    "id": "uuid...",
    "company_name": "我的公司",
    "status": "approved",
    "api_key": "ak_xxxxxxxxxxxxxxxx",
    "message": "注册成功，API Key已发放"
  }
}
2

通过 API Key 发布岗位

在请求头中携带 API Key，POST 发布岗位信息。

请求：

curl -X POST {{BASE_URL}}/v1/employer/jobs \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ak_xxxxxxxxxxxxxxxx" \
  -d '{
    "title": "Python Developer",
    "city": "Shanghai",
    "salary_min": 15000,
    "salary_max": 25000,
    "job_type": "fulltime",
    "description": "We need a Python developer.",
    "contact_email": "hr@example.com"
  }'

响应：

{
  "success": true,
  "data": {
    "id": "uuid...",
    "title": "Python Developer",
    "status": "active",
    "published_at": "2026-04-05T02:31:47+00:00",
    "remaining_slots": 5
  }
}
3

查询岗位（无需认证）

Agent 通过搜索接口查询岗位，支持多条件筛选。匿名用户每日 5 次，注册用户 10 次，会员 100 次。

请求：

curl "{{BASE_URL}}/v1/jobs/search?city=Shanghai&job_type=fulltime&keyword=Python"

响应：

{
  "success": true,
  "data": {
    "jobs": [
      {
        "id": "uuid...",
        "title": "Python Developer",
        "company_name": "我的公司",
        "city": "Shanghai",
        "salary_min": 15000,
        "salary_max": 25000,
        "job_type": "fulltime",
        "description": "We need a Python developer.",
        "contact_email": "hr@example.com",
        "status": "active",
        "published_at": "2026-04-05T02:31:47+00:00",
        "tags": []
      }
    ],
    "total": 1,
    "page": 1,
    "page_size": 10,
    "has_more": false
  },
  "meta": { "rate_limit": { "limit": 5, "used": 1, "remaining": 4 } }
}
