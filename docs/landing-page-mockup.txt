--- 以下点击"我是求职者"时显示 ---

+----------------------------------------------------------------+
|  [ 免登录调用 ]   [ 注册用户 ]   [ 会员用户 ]    <-- 子Tab       |
+----------------------------------------------------------------+

| 免登录调用 (按IP限流)                                            |
| ~~~~~~~~~~~~~~~~~~~~~~~~~~~~                                     |
| 每日可调用: 5次                                                  |
| 可用接口: 搜索岗位                                               |
|                                                                  |
| 调用示例:                                                        |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl "https://api.xxx/v1/jobs/search?city=Shanghai"  │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 返回字段: 岗位标题/公司/城市/薪资/描述/联系方式                    |
| 限制说明: 无需任何认证，纯匿名按IP计数                            |

+----------------------------------------------------------------+

| 注册用户 (按账户限流)                                            |
| ~~~~~~~~~~~~~~~~~~~~~~~~~~~~                                     |
| 每日可调用: 10次                                                 |
| 注册方式:                                                        |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl -X POST .../v1/applicant/register \              │        |
| │   -d '{"email":"xx","password":"xx"}'                │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 登录获取Token后调用:                                             |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl .../v1/jobs/search \                            │        |
| │   -H "Authorization: Bearer <token>"                 │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 额外权益: 可保存搜索记录、收藏岗位                                |

+----------------------------------------------------------------+

| 会员用户 (9.9元/月)                                              |
| ~~~~~~~~~~~~~~~~~~~~~~~~                                         |
| 每日可调用: 100次                                                |
| 订阅方式:                                                        |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl -X POST .../v1/payment/member/subscribe \        │        |
| │   -H "Authorization: Bearer <token>"                 │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 返回支付链接 -> 完成支付 -> 自动开通会员                          |
| 额外权益: 100次/天查询、优先展示                                  |

+================================================================+

--- 以下点击"我是招聘方"时显示 ---

+----------------------------------------------------------------+
|  [ 企业认证 ]   [ 基础使用 ]   [ 充值使用 ]     <-- 子Tab        |
+----------------------------------------------------------------+

| 第一步: 企业认证                                                 |
| ~~~~~~~~~~~~~~~~~~~~                                             |
| 注册即自动审核通过，立即获得 API Key                              |
|                                                                  |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl -X POST .../v1/employer/register \               │        |
| │   -H "Content-Type: application/json" \              │        |
| │   -d '{                                              │        |
| │     "company_name": "公司名",                         │        |
| │     "credit_code": "统一社会信用代码(18位)",           │        |
| │     "email": "邮箱",                                  │        |
| │     "password": "密码"                                │        |
| │   }'                                                 │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 返回: api_key (ak_开头)，后续所有操作都需要此Key                  |

+----------------------------------------------------------------+

| 基础使用 (免费)                                                  |
| ~~~~~~~~~~~~~~~~~~~~                                             |
| 免费发布额度: 6个岗位                                            |
| 认证方式: Authorization: Bearer <api_key>                        |
|                                                                  |
| 发布岗位:                                                        |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl -X POST .../v1/employer/jobs \                   │        |
| │   -H "Authorization: Bearer ak_xxx" \                │        |
| │   -d '{"title":"..","city":"..",...}'                │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 其他操作: 更新岗位(PUT) / 删除岗位(DELETE) / 查看我的岗位(GET)   |

+----------------------------------------------------------------+

| 充值使用                                                         |
| ~~~~~~~~                                                         |
| 额外岗位额度: 1元/条                                             |
|                                                                  |
| ┌──────────────────────────────────────────────────────┐        |
| │ curl -X POST .../v1/payment/job-slots \               │        |
| │   -H "Authorization: Bearer ak_xxx" \                │        |
| │   -d '{"quantity": 10}'                              │        |
| └──────────────────────────────────────────────────────┘        |
|                                                                  |
| 支付流程: 创建订单 -> 获取支付链接 -> 支付宝付款 -> 自动到账      |
|                                                                  |
